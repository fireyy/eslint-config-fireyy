# ESLint config for fireyy

Work with [eslint](https://eslint.org)

## Features

Use [Standard](https://standardjs.com) style

- Indent with 2 spaces
- no semicolon
- Lint code blocks in Markdown
...

## Install

```bash
$ yarn add eslint eslint-config-fireyy babel-eslint -D
```

## Usage

In ESLint:

```js
{
  "eslintConfig": {
    "extends": ["fireyy"]
  }
}
```
